Автоматическая экстракция химических данных из научных PDF

ChemX · домен SelTox

Превзойти single-agent baseline в 4.2×

Проблема

  • Химия — наука, перегруженная данными, но результаты заперты в неструктурированных PDF.
  • Бенчмарк ChemX (NeurIPS 2025): извлечь структурированные таблицы из статей.
  • Главная сложность — данные часто в таблицах, графиках и рисунках, а не в тексте.

Задача: система экстракции, которая бьёт опубликованный baseline хотя бы на одном домене.

Почему SelTox

  • Наночастицы серебра: антимикробная активность и токсичность, 3240 строк.
  • Самая низкая планка baseline среди всех доменов: Macro-F1 = 0.0454.
  • 23 целевых поля, без SMILES → проще, чем малые молекулы.
  • 6 полей у baseline в нуле → много «свободного» хедрума.

Сначала — понять метрику

ChemX-метрика: exact-string multiset по строкам внутри статьи, Macro-F1 по 23 полям.

  • Скорятся только 23 целевых поля, не метаданные.
  • Формат — per-field: float64 → "10.0", бинарные → 0/1.
  • Построили доверенный evaluator — меряем себя честно (all-65 lowercase).

Владение метрикой = половина успеха.

Архитектура (один проход)

PDF (+ supplementary)
  │
  ▼ 1. Parse — текст + таблицы → markdown, страницы → PNG
  ▼ 2. Context — весь документ (262k токенов), без BM25
  ▼ 3. Extract — один schema-driven вызов LLM
  ▼ 4. Normalize — под точный формат gold
  ▼ 5. Submission — prior(65) + извлечённые строки
  ▼ 6. Evaluate — доверенный ChemX-evaluator

Принципы: mono-agent, abstention-first, exact-string под gold.

Идея 1 — калиброванный prior

Мажоритарные дефолты по полям с 0% пропусков:

np=Ag · method=MIC · shape=spherical · precursor=AgNO3 · mdr=0 …

Floor без единого вызова LLM:

0.1357  vs baseline 0.0454  →  ×3 уже здесь

Идея 2 — LLM-экстракция

  • Модель Qwen3.6-35B (Yandex AI Studio), контекст 262k, OpenAI-совместимый API.
  • Один вызов на статью: весь документ → {samples:[…]}.
  • Одна строка на (бактерия × наночастица).
  • Abstention-first: нет в тексте → null, не выдумываем.

0.1357 → 0.1863

Идея 3 — нормализация под gold

LLM пишет свободно — gold хранит канон. Маппим:

  • method: «Broth microdilution assay» → MIC, «Agar well diffusion» → ZOI
  • precursor: «Silver nitrate» → AgNO3, прочее → NOT_DETECTED

Возвращает истинные ZOI/MBC, которые MIC-only prior пропускал.

0.1863 → 0.1887 → 0.1921
